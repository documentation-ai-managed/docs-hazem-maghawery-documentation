## Rate Limiting

<Callout kind="danger">API requests are rate limited to prevent abuse. Monitor your usage and implement retry logic.</Callout>

<ExpandableGroup>
  <Expandable title="Rate Limits" default-open="true">

## Error Handling

<CodeGroup tabs="JavaScript,Python">
  ````javascript
  try {
    const response = await fetch('/api/docs', {
      headers: { 'Authorization': `Bearer ${API_KEY}` }
    });
    
    if (!response.ok) {
      if (response.status === 429) {
        // Rate limited - wait and retry
        await new Promise(resolve => setTimeout(resolve, 60000));
        return retryRequest();
      }
      throw new Error(`API Error: ${response.status}`);
    }
    
    const data = await response.json();
    return data;
  } catch (error) {
    console.error('API request failed:', error);
    // Handle network errors, timeouts, etc.
  }
  ````
  ````python
import time

def make_api_request(url, api_key, max_retries=3):
    headers = {'Authorization': f'Bearer {api_key}'}
    
    for attempt in range(max_retries):
        try:
            response = requests.get(url, headers=headers)
            
            if response.status_code == 429:
                # Rate limited
                retry_after = int(response.headers.get('Retry-After', 60))
                time.sleep(retry_after)
                continue
            
            response.raise_for_status()
            return response.json()
        except requests.RequestException as e:
            if attempt == max_retries - 1:
                raise e
            time.sleep(2 ** attempt)  # Exponential backoff
    
    return None
  ````

## SDK Examples

<Expandable title="Node.js SDK" default-open="false">
  ```javascript
  const { DocumentationClient } = require('@hazemmaghawery/docs-sdk');
  
  const client = new DocumentationClient({
    apiKey: 'your-api-key',
    baseUrl: 'https://api.docs.example.com'
  });
  
  // Create a document
  const doc = await client.documents.create({
    title: 'New API Guide',
    content: '# Introduction\nThis is a new guide...'
  });
  
  console.log('Created document:', doc.id);
  ```
</Expandable>

<Expandable title="Python SDK" default-open="false">
  ```python
  from hazemmaghawery_docs import DocumentationClient
  
  client = DocumentationClient(
      api_key='your-api-key',
      base_url='https://api.docs.example.com'
  )
  
  # List documents
  docs = client.documents.list()
  for doc in docs:
      print(f"Document: {doc.title} (ID: {doc.id})")
  ```
</Expandable>

## Best Practices

Use environment-specific API keys for different deployment stages. Implement comprehensive logging for API requests and responses. Regularly rotate API keys for security.

Test integrations thoroughly in staging environments before deploying to production. Monitor API usage and set up alerts for unusual activity.

Document your integrations clearly for future maintenance and troubleshooting.
